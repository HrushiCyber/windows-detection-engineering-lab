# Network Telemetry

Endpoint detection platforms monitor network activity generated by processes.

---

# Network Connection Events

Common telemetry attributes include:

• source process
• destination IP
• destination port
• protocol
• connection timestamp

---

# Detection Opportunities

Network telemetry can reveal:

• command and control activity
• data exfiltration
• suspicious outbound connections

---

# Example Detection

Processes communicating with rare external domains may indicate C2 infrastructure.
