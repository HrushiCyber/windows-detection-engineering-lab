# Command and Control Tradecraft

Command and control (C2) infrastructure allows attackers to communicate with compromised systems.

---

# Common C2 Techniques

### HTTP/HTTPS Beaconing

Malware periodically communicates with attacker servers.

---

### DNS Tunneling

Attackers encode data in DNS queries to exfiltrate information.

---

### Encrypted C2 Channels

Attackers frequently use encrypted channels to hide network traffic.
