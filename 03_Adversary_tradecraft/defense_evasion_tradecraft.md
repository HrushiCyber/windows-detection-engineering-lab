# Defense Evasion Tradecraft

Attackers use defense evasion techniques to avoid detection.

---

# Common Defense Evasion Techniques

### Process Injection

Malicious code is injected into legitimate processes.

---

### AMSI Bypass

Attackers disable or bypass the Antimalware Scan Interface to evade script scanning.

---

### DLL Search Order Hijacking

Attackers place malicious DLLs where applications will load them instead of legitimate ones.

---

### Parent Process Spoofing

Attackers manipulate parent process IDs to disguise execution chains.
